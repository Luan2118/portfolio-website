# Luan Le Portfolio

Personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

This portfolio presents my frontend projects, technical skills, and contact information.

## Features

* Responsive single-page layout
* Project cards with screenshots, descriptions, tech stack, live demo links, and GitHub links
* Tech stack section
* Contact section
* Fixed navigation with section links

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
